Panoramica archivio

Terremoto

  • Controlla ferite e pericoli immediati.
  • Resta in allerta per le scosse.

Alluvione

  • Sali in zone elevate.
  • Evita l'acqua.

Uragano / Tornado

  • Riparati lontano dalle finestre.
  • Attendi il via libera.

Incendio boschivo

  • Allontanati dal fumo controvento.
  • Preparati a evacuare.

Incidente in centrale elettrica

  • Rimani al chiuso.
  • Segui le istruzioni alla radio.

Cyberattacco / Guasto infrastrutturale

  • Usa note offline e radio.
  • Risparmia batteria.

Bombardamento

  • Riparo basso e interno.
  • Evita le finestre.

Sabotaggio / Attivita sospetta

  • Evita aree colpite.
  • Tieni distanza dagli oggetti.

Blocco / Movimento limitato

  • Raziona le scorte.
  • Limita gli spostamenti.

Tempesta di ghiaccio

  • Rimani in casa.
  • Evita di viaggiare.

Calore estremo

  • Idratati spesso.
  • Raffreddati regolarmente.

Freddo estremo

  • Vestiti a strati.
  • Riscalda una stanza.

Interruzioni pianificate / di emergenza

  • Usa torce.
  • Risparmia energia.

Crisi economica

  • Dai priorita all'essenziale.
  • Segui gli avvisi ufficiali.
